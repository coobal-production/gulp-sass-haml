# gulp-sass-haml
gulpでsass（compass-spriteのみ）とhamlを使用するhtml5テンプレート

## package.json

- browser-sync: "^2.7.7"
- gulp: "^3.9.0"
- gulp-autoprefixer: "^2.3.1"
- gulp-compass: "^2.1.0"
- gulp-concat: "^2.5.2"
- gulp-cssmin: "^0.1.7"
- gulp-plumber: "^1.0.1"
- gulp-rename: "^1.2.2"
- gulp-ruby-haml: "0.0.4"
- gulp-uglify: "^1.2.0"

## sass（compass-sprite）

- gulp-compass

rubyおよびsass、compassをインストールしてください。

## haml

- gulp-ruby-haml
- padrino-helpers

rubyおよびhaml、padrinoをインストールしてください。
